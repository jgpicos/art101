In this screenshot, you can see how I created a pause on the else statement in the code. That else is only run if the i’th number cannot be divided by any of the four options. Using the debugger tool (and a lot of clicking) we can find out which numbers can’t be divided. It also adds comments for each line telling us EXACTLY what is equal to what. This is really neat because we know a couple languages and haven’t even bothered to use this feature.
